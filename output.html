Penjelasan cara menampilkan hasil di HTML dan browser

💬

console.log()

Mengirim pesan ke konsol browser. Bukan tampilan langsung di halaman web.

console.log("Halo dunia");
⚠️

alert()

Menampilkan kotak popup di browser. Pengguna harus menutupnya untuk melanjutkan.

alert("Halo dunia");
🧾

document.write()

Menulis langsung ke halaman HTML saat dokumen sedang dimuat.

document.write("Halo dunia");
✍️

element.innerHTML

Memasukkan atau mengubah konten HTML di dalam elemen yang sudah ada.

document.getElementById("hasil").innerHTML = "Halo";
📝

element.textContent

Menuliskan teks saja ke dalam elemen, tanpa menafsirkan HTML.

document.getElementById("hasil").textContent = "Halo";
🔗

Gabungan Output

Biasanya kita pakai console.log untuk debugging dan innerHTML / textContent untuk menampilkan hasil ke user.
