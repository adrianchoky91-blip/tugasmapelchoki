Array Toko Mini

Belajar buat, update, dan hapus data array menggunakan model toko.

📦

Buat Array Produk

Gunakan array objek untuk menyimpan data barang di toko.

const products = [
  { id: 1, name: "Beras", price: 12000, stock: 10 },
  { id: 2, name: "Gula", price: 15000, stock: 8 },
  { id: 3, name: "Minyak", price: 18000, stock: 5 }
];
✏️

Update Sebagian

Ubah harga atau stok satu produk tanpa membuat array baru.

// Ubah harga produk ID 2
const product = products.find(item => item.id === 2);
if (product) {
  product.price = 14000;
}
🗑️

Hapus Satu Produk

Gunakan filter untuk menghapus item dari array.

const newProducts = products.filter(item => item.id !== 3);
🧹

Hapus Semua Produk

Reset array menjadi kosong saat toko tutup atau stok habis.

products.length = 0;
// atau
// products = [];
🔁
