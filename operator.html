JS

Kalkulator Operator

Perbandingan dan operator logika pada JavaScript.

=

Operator Perbandingan

Bandingkan dua nilai menggunakan ==, ===, !=, dan !==.

>

Lebih Besar / Kecil

Gunakan >, <, >=, dan <= untuk membandingkan angka.

&&

Operator Logika

Kombinasikan beberapa perbandingan dengan && dan ||.

!

Negasi

Gunakan ! untuk membalik nilai boolean.

Hitung Perbandingan

Masukkan dua nilai, pilih operator, lalu lihat hasil dan penjelasannya.
